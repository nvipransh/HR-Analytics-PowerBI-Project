# HR-Analytics-PowerBI-Project
Conducted an HR analytics project using Microsoft Excel and Microsoft Power BI to analyze employee data and identify factors influencing attrition and workforce trends.

# Tools Used

The project was completed using:
- Microsoft Excel for initial data handling
- Microsoft Power BI for data cleaning, transformation, analysis, and dashboard creation

# Data Collection

- The dataset consisted of employee-related information, including attributes such as demographics, job roles, department details, performance indicators, and        attrition status. The data was provided in a structured format suitable for analysis.

# Data Cleaning (Power BI – Power Query)

Data cleaning was primarily performed using Power BI’s Power Query Editor:
- Removed duplicate records to ensure data integrity
- Handled missing and null values appropriately
- Standardized categorical fields such as department, job role, and attrition status
- Cleaned and formatted text fields to maintain consistency
- Corrected data types for accurate processing and analysis

# Data Transformation

The dataset was transformed within Power BI to make it analysis-ready:
- Applied data grouping and categorization techniques
- Used Power Query for shaping and restructuring the dataset
- Built a structured data model to support efficient analysis

# Data Analysis

The analysis focused on identifying workforce trends and patterns:
- Evaluated employee attrition across departments, roles, and demographics
- Analyzed factors like job satisfaction, performance, and working conditions
- Explored relationships between different HR variables

# Data Visualization (Power BI Dashboard)

An interactive dashboard was developed using Power BI:
- Designed visuals such as bar charts, pie charts, and KPI indicators
- Used slicers and filters for dynamic user interaction
- Structured the dashboard for clear and intuitive understanding
- Enabled drill-down features for deeper insights

# Insights Generation

From the analysis:
- Identified key drivers influencing employee attrition
- Observed trends across different employee segments
- Highlighted areas requiring improvement in workforce management
- Provided insights to support better HR decision-making

# Conclusion
This project demonstrates the effective use of Power BI for end-to-end data analysis, including data cleaning, transformation, and visualization. The approach enabled meaningful insights into employee behavior and organizational trends, supporting data-driven decision-making.
